# Overview

This framework provides a GUI for CreateJS animations. Examples are:

- Different sliders
- Buttons
- Numeric stepper
- And more ...

It simplifies the process of animating specific graphical elements, such as:

- Spline curves
- Coordinate systems
- Vector arrows
- And more...
